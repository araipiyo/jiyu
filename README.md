# 自由主義のメモ

「個人の自由を制限できるのは、その行為が他者に危害を加える場合に限られる」とミルは著書「自由論」のなかで記しました。これは多様な人々が暮らす社会において、少数派の権利を守る唯一の道しるべと言っても過言ではありません。多数派は、つねに自分たちの価値観を、暴力を使って、少数派に押し付けようとしてきたからです。

## 刑罰へ反対しよう

刑罰とは、国家による個人への暴力に他なりません。その暴力は「他者への危害」を防ぐために、ほかに方法が無い場合のみ、例外的に、最小限度で許されると考えるべきです。

とくに刑務所やその他の拘禁施設はあまりに過酷です。たとえ犯罪者であっても、そのようなところに人間を閉じ込めることが許されるとは、とうてい考えられません。そのことに多くの人が気づかないのは恐ろしいことです。

全ての刑罰をただちに廃止できるとは私も信じてはいませんが、ゼロから考え直すことが必要です。中世の死刑や拷問中心の刑罰を置き換えた、近世の監獄制度を21世紀になっても、当たり前のように受け入れてるのは理解に苦しむところです。

## 民主主義への盲信をやめよう

民主主義(政治)は、あくまで話し合いがどうしても決着しない場合に、やむを得ずに多数決や政治工作で物事を決めるための最後の手段です。

民主主義の上位の価値として、自由・人権・法治があることが大前提であり、それを明文化したものが憲法です。

民主主義で決めたのだから、何でも許されるという考え方は間違いです。

近年の日本では、議員立法の暴走があまりに目に余ります。本来であれば、常設の憲法裁判所を作り、すべての新しい法律を厳しく憲法裁判所で審査すべきではないでしょうか。しかしそれを実現することは難しいので、われわれ市民がまとまって声を上げていきましょう。
